## Deployment

The IDS can be deployed in:

- Local SOC environments
- Enterprise servers
- Cloud virtual machines
- Docker containers

Supported deployment modes:

1. Synthetic Traffic Simulation
2. Dataset Replay Detection
3. Live NIC Packet Capture
4. PCAP Analysis

Requirements:
- Python 3.11
- Npcap (Windows)
- TensorFlow
- XGBoost
- Streamlit

Launch:

streamlit run dashboard.py
